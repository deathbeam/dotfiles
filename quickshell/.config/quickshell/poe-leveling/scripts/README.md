# PoE Leveling Overlay Scripts

## import_exile_data.sh

Imports PoE1 leveling data from Exile-UI repository.

```bash
./scripts/import_exile_data.sh
```

---

## toggle.sh

Toggle the overlay on/off.

```bash
./scripts/toggle.sh
```

---

## test_overlay.sh

Integration test that simulates zone changes.

```bash
./scripts/test_overlay.sh
```
